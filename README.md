- Anne-Charlotte :
  Pour la réalisation de la page "Boutique", j'ai utilisé le code HTML5 de manière logique et utilisé le framework Bootstrap pour la mise en page responsive.
  La présentation des œuvres via les balises HTML5 <figure> et <figcaption> pour associer les images à leur légende, au cas où il y aurait une erreur d'affichage.
La mise en page adaptive est assurée grâce au système de grille (col-12 vers col-md-4) et à la classe .img-fluid. Cela garantit un contenu bien hierarchisé et un affichage fluide et lisible, à partir de n'importe quel écran.
